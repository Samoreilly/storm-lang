# Storm - procedural language

1. The entry point is a tempest procedure

```
    void tempest() {

    }
```

--- Storm Language Rules ---
1. Statements must end with a semicolon ```;```
2. Blocks are enclosed in braces ```{}``` 
3. All keywords must be lowercase
4. Comments must be enclosed inside of 2 pairs of square brackets ```[[]]```



--- Type rules ---
1. Uninitializied int and double variable types will be set to 0 and 0.0
2. Const cannot be modified
